# osu!Remod
Allows you to change mods on osu! replays.
Usage: osu!Remod.exe <Usage: osu!Remod.exe <input OSR file> <output OSR file> (Mods)

You can specify as many mods as you want, and here are their abbreviations:

Nofail: NF

Easy: EZ

NoVideo: NV

Hidden: HD

Hardrock: HR

Sudden Death: SD

Doubletime: DT

Relax: RX

Halftime: HT

Nightcore: NC (This is only responsible for the bass. Use with DT for the typical effect)

Flashlight: FL

Auto: AUTO

SpunOut: SO

AutoPilot: AP

Perfect: PF

1 Key: 1K

2 Keys: 2K

3 Keys: 3K

4 Keys: 4K

5 Keys: 5K

6 Keys: 6K

7 Keys: 7K

8 Keys: 8K

9 Keys: 9K

Fade In: FI

Random: RD

Cinema: Cinema

Target Practice: TP

Co-op: COOP
